Computer Vision
(2024)

This project consists labelled data to train as 'cars' with boundary box classification for images and ‘non car’ images. Trained on unlabelled dataset to classify cars using Histogram of Gradient features and Support Vector Machine. Employed two YOLO models with varying architectures to detect vehicles from a predefined dataset (Coco dataset). Utilised Bounding box prediction to derive objectness score based on pixel values relative to lighting, to calculate objectness score.
